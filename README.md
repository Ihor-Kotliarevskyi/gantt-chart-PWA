# 📅 Ганtt Про - PWA версія

Ваш додаток тепер готовий як **Progressive Web App (PWA)**! 🎉

## 📋 Що змінилося?

✅ **Офлайн режим** — працює без інтернету  
✅ **Встановлення на комп'ютер/телефон** — як звичайна програма  
✅ **Швидша загрузка** — кешування даних  
✅ **Синхронізація** — дані зберігаються локально  

## 🚀 Як встановити?

### 🖥️ На Windows/Mac/Linux

1. **Відкрийте файл у браузері:**
   - Подвійний клік на `Ганtt_Про_v4_1.html` або
   - Перетягніть файл в браузер

2. **Встановіть як app:**
   - Chrome/Edge: Натисніть 📱 значок у адресному рядку → "Встановити додаток"
   - Safari (Mac): Меню → "Додати до Dock"
   - Firefox: Меню ≡ → "Встановити додаток"

3. **Готово!** Додаток буде в стартовому меню / панелі завдань

### 📱 На телефоні (Android/iOS)

**Android:**
1. Відкрийте файл у Chrome
2. Меню ⋮ → "Встановити додаток"

**iPhone/iPad:**
1. Відкрийте файл у Safari
2. Натисніть Поділитися → "На екран Домівки"

## 📁 Необхідні файли

Всі файли повинні бути в одній папці:
```
📁 Папка
  📄 Ганtt_Про_v4_1.html    (основний файл)
  📄 sw.js                   (Service Worker)
  📄 manifest.json           (конфіг PWA)
  📄 README.md               (цей файл)
```

## 💾 Особливості

- **Локальне зберігання** — дані зберігаються у браузері
- **Мультипроекти** — кілька проектів одночасно
- **Імпорт/Експорт** — JSON, Excel (CSV)
- **Фінальна аналітика** — графіки, звіти
- **Управління командою** — підрядники, категорії

## 🔧 Як працює PWA?

| Функція | Опис |
|---------|------|
| **Manifest** | Файл `manifest.json` описує додаток для ОС |
| **Service Worker** | `sw.js` дозволяє роботу офлайн |
| **Кеш** | Статичні файли кешуються при першому відвідуванні |
| **Синхронізація** | При возвращенні онлайну дані синхронізуються |

## 🌐 Розміщення на сервері (опціонально)

Для доступу з будь-якого місця:

1. Завантажте всі три файли на веб-сервер (з HTTPS)
2. Відкрийте URL в браузері
3. Встановіть як додаток

## ⚠️ Важливо

- **HTTPS** потрібен при розміщенні на сервері
- **Локальне відкриття** (`file://`) працює лише в Chrome/Edge
- **Дані** зберігаються окремо для кожного браузера
- **Резервна копія** — регулярно експортуйте JSON

## 🐛 Трішутинг

**"Service Worker не реєструється"**
- Перевірте, що `sw.js` в одній папці з HTML
- Спробуйте в браузері Chrome (найкраще підтримується)

**"Не встановлюється як додаток"**
- PWA має працювати на HTTPS (офлайн на HTTP може не спрацювати)
- Перезавантажте сторінку та повторіть спробу

**"Дані зникли при оновленні браузера"**
- Дані зберігаються в `localStorage`
- Експортуйте важливі проекти в JSON
