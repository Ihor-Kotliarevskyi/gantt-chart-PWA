## 🎯 Швидкий старт

### 1. Локальне тестування

**Windows:**
```batch
start-server.bat
```

**Mac/Linux:**
```bash
bash start-server.sh
```

Потім: **http://localhost:8000**

### 2. Встановлення як додаток

1. Натисніть значок у адресному рядку браузера 📱
2. Виберіть "Встановити"
3. Готово!

### 3. На публічному сервері

1. Завантажте всі файли на HTTPS сервер
2. Переконайтеся, що `.htaccess` присутній (для Apache)
3. Обновите посилання на `manifest` в HTML

- ✅ Офлайн режим
